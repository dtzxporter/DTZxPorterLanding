## 👉 Usage:
Read the new game tools tutorial: [Game Tool Tutorial](https://dtzxporter.com/game-tools-tutorial).

> **IMPORTANT:** Both the tool and the game **require** the entirety of the game folder to be intact and must not be modified in any way. Rime loads all of the files in the game folder at once after picking a file to load.

## ⚙️ Ripping:
- The **.npk** files contain various encoded assets that Rime can export; and, as of now the currently supported assets are:
  - Textures as [Dds, Png, Tiff, Tga]
  - Models as [Cast, OBJ, XNALara, SMD, CoD XModel, Maya, FBX].
  - Animations as [Cast].
  - Raw Files as-is.
